# 1. Các bước viết unit test:

- Identify all possible case
- Specify parameters and expected results for each case
- Write test
- Execute test
- Evaluation and assessment of results

# 2. Các thành phần cơ bản có trong 1 unit test:

- Test suit
- Block test
- Test case
- Action
- Assert
